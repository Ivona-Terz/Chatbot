# XYZ Beauty Website
This project is a fictional makeup brand website developed as part of a group assignment.
## Features
- Responsive website design
- Product showcase section
- Rule-based chatbot for user interaction
## Technologies Used
- HTML
- CSS
- JavaScript
## Chatbot Functionality
The chatbot is rule-based and responds to user input using keyword detection (e.g. lipstick, foundation, shipping).
## Purpose
The project demonstrates the design and implementation of a chatbot integrated into a front-end website environment.
